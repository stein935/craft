Command: start

Usage: 
  craft start -n <server_name> [options]     Usage details: Run '$ craft start -h'

Required:
  -n <server_name>     Name of server to start       

Options:
  -m                   Enables monitor. Automatically restart server when it fails
  -f <minutes>         Frequency (in minutes) of server monitor. Setting this value will
                       also enable -m (monitor)
  -d                   TODO _daemon_
  -p                   Writes active players on server 
                       to players.txt
