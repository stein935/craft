Command: command

Usage: 
  craft command -n <server_name> -c <command [ options ]>   Usage details: Run '$ craft command -h'

Required:
  -n <server_name>                Name of server to command      
  -c <command [ options ]>        Minecraft server command 

Options:
  -p                              Writes active players on server 
                                  to players.txt
